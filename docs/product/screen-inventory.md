# Screen Inventory

## Approved top-level screens

### S01 — Portfolio Swimlane Home
Purpose: Portfolio-wide visibility workspace for workload, ghost load, risk/warnings, and navigation into diagnosis/review.

### S02 — Planning Setup
Purpose: Compact setup/readiness surface for source readiness and capacity-input readiness.

### S03 — Resource Detail
Purpose: Focused single-resource diagnostic workspace with recommendation consumption and warning context.

### S04 — Delta Review
Purpose: Formal review surface for draft-vs-approved changes, selective acceptance, connected-set handling, and explicit activation.

### S05 — Planning Warnings Workspace
Purpose: Dedicated warning/trust review workspace with navigation back to owning workflows.

## Approved embedded interactions

### D01 — Swimlane Task Drill-Down Drawer
Purpose: Task-level detail launched from a swimlane segment on S01.

### M01 — Connected Change Set Modal
Purpose: Dependency-safe acceptance surface launched from S04 when isolated acceptance is unsafe.

## Core navigation model
- S01 is the main planning home.
- S02 is the readiness/setup gate.
- S03 is the focused resource diagnosis surface.
- S04 is the formal review/accept/activate surface.
- S05 is the dedicated warning/trust review surface.
